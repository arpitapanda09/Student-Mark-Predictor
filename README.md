## 📌 Project Overview

Student Mark Predictor is a Machine Learning project developed during my internship training at the Government Skill Development Institute (SDI), Bhubaneswar.

The primary objective of this project is to predict student marks based on academic performance data using Machine Learning techniques. This project demonstrates the complete Machine Learning lifecycle, including data collection, preprocessing, exploratory data analysis, model training, evaluation, and prediction.

The project provides practical insights into educational data analysis and predictive modeling while helping develop hands-on experience in Data Science and Machine Learning.

---

## 🎯 Problem Statement

Educational institutions generate a large amount of student performance data. Analyzing this data manually can be time-consuming and inefficient.

The goal of this project is to develop a Machine Learning model that can predict student marks based on available academic information, helping educators and students better understand performance trends.

---

## 🎯 Objectives

- Analyze student academic performance data
- Perform data preprocessing and cleaning
- Explore data using visualization techniques
- Train and evaluate Machine Learning models
- Predict student marks accurately

## 📊 Dataset Information

The dataset contains information related to student academic performance.

Example Features:

- Study Hours
- Attendance
- Previous Scores
- Assignments
- Internal Marks
- Final Marks

The dataset was preprocessed before training the Machine Learning model.

---

## 🔄 Project Workflow

### 1️⃣ Data Collection

The dataset containing student academic information was collected and imported into Python for analysis.

### 2️⃣ Data Preprocessing

The following preprocessing techniques were applied:

- Handling missing values
- Data cleaning
- Feature selection
- Data transformation

### 3️⃣ Exploratory Data Analysis (EDA)

EDA was performed to understand the dataset and identify meaningful patterns.

Activities included:

- Statistical summaries
- Correlation analysis
- Data visualization
- Feature relationship analysis

### 4️⃣ Model Training

The dataset was divided into training and testing sets.

Machine Learning algorithms were applied to train the predictive model.

### 5️⃣ Model Evaluation

The model performance was evaluated using standard metrics.

Evaluation metrics included:

- MAE (Mean Absolute Error)
- MSE (Mean Squared Error)
- RMSE (Root Mean Squared Error)
- R² Score

### 6️⃣ Prediction

The trained model predicts student marks based on input academic data.
